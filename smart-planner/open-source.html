Open Source Licenses

Smart Planner uses open-source software and libraries to provide app features, improve reliability, and support development.

License notices

Open-source license information is available inside the app through the Open Source Licenses screen. This screen may include license notices for Flutter, Dart, Android libraries, and other packages used by Smart Planner.

How to view

In the app, go to: Settings > Legal > Open Source Licenses.

Implementation note

Where possible, Smart Planner uses Flutter's built-in license page to display package license information.

Contact

If you have questions about open-source licenses used in Smart Planner, contact us at support@feathly.com.
