Support

Questions, feedback, and early release support for Smart Planner.

Contact

Email: support@feathly.com

Please include

  • App version
  • Device model
  • Android version
  • A short description of the issue
  • Steps to reproduce the issue, if possible
  • Screenshots, if helpful

Purchase support

If you contact us about premium access, please first try Restore Purchase using the same Google Play account. Then include the approximate purchase date when contacting support.

Response time

Smart Planner is an early-stage app, so response time may vary. We will try to respond as soon as practical.
