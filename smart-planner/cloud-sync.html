Cloud Sync Notice

Smart Planner v1 is local-first. Cloud sync and backup are not included in the first release.

Your Smart Planner data is stored locally on your device in v1. Please export or back up data before changing devices or uninstalling.

What is not included in v1

  • Feathly cloud account
  • Real-time multi-device sync
  • Cloud backup and restore
  • Server-side loop storage

Possible future cloud features

Future versions may introduce cloud backup, multi-device sync, cloud restore, account-based premium migration, or shared server storage.

Future pricing

Because cloud features create ongoing server costs, future cloud sync or cloud backup may require a separate fee or subscription. We will not charge for cloud features without clear notice.
