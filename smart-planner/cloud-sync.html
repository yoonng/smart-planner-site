Cloud Sync Notice

Last updated: 2026-06-16. Smart Planner v1 is designed as a local-first app.

In Smart Planner v1, your loop data is stored locally on your device. Real-time cloud sync, cloud backup, multi-device sync, and Feathly cloud accounts are not included in the first release.

1. Current data storage

In Smart Planner v1, your loop data is stored locally on your device. This may include loops, notes, categories, reminder schedules, review records, routine records, and app settings.

Smart Planner v1 does not provide real-time cloud sync, cloud backup, multi-device sync, or a Feathly cloud account.

2. Device changes and backups

Because Smart Planner v1 is local-first, changing devices, uninstalling the app, clearing app data, or losing your device may cause local data loss unless you have exported or backed up your data.

If export or import features are available, we recommend using them before changing devices or resetting your device.

3. Future cloud features

Feathly may add cloud-based features in the future, such as real-time sync across devices, cloud backup and restore, multi-device history, server storage, account-based premium migration, and other server-based features.

These features require ongoing server operation, storage, maintenance, and security costs.

4. Possible additional cloud fee

A Smart Planner lifetime purchase covers the local premium features described at the time of purchase. It does not guarantee that all future cloud or server-based features will be free forever.

If Feathly adds real-time sync, cloud backup, server storage, or other server-based features in the future, those features may require a small additional fee or subscription to cover ongoing server costs.

5. Clear notice before paid cloud features

Feathly will clearly explain any additional cloud fee before users choose to use a paid cloud feature.
